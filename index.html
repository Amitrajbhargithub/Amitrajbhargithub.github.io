Who I Am

I'm a passionate full-stack web developer who loves building performant, maintainable applications that solve real problems. With expertise in both frontend and backend technologies, I create seamless user experiences backed by robust server architectures.

My journey in web development has led me to specialize in:

  • Modern Web Applications: Building responsive, interactive interfaces with Vue.js, Nuxt.js, and React

  • E-commerce Solutions: Developing shopping platforms with Shopify and custom integrations
